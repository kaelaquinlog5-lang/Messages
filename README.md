To get a message code:

1. Write a message - Fill in your name and write your message in the "Write Your Message" section
2. Click "Save Message" - After you submit your message, a unique 6-letter code will automatically appear
3. Your code shows up - Look for the gray box that appears saying "Your message code:" with letters like "ABC123"
4. Share the code - Give this code to the teachers of your choice.

To view someone else's message:

1. Get their code - Ask them for their 6-letter message code
2. Enter the code - Type it in the "Enter message code" box in the "View Messages" section
3. Click "View Message" - Your message will appear!
